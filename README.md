# Experiment 33: Count Vowels, Consonants, Digits

## Problem Statement

Write a program to count the number of **vowels**, **consonants**, **digits**, and **special characters** in a given string.

## Input Format

* A single string $S$.

## Output Format

* Print four space-separated integers in this specific order:
  `Vowels Consonants Digits Special`

### Example 1

**Input:**

```text
JavaLab@2025
```

**Output:**

```text
3 4 4 1
```

### Explanation
* **Vowels:** a, a, a (3)
* **Consonants:** J, v, L, b (4)
* **Digits:** 2, 0, 2, 5 (4)
* **Special:** @ (1)
* Total: 3 + 4 + 4 + 1 = 12 characters.
